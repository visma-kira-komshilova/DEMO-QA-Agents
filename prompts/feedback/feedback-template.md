# Code Review Accuracy Report Template

Output format for aggregated accuracy reports. For analysis logic, see `feedback-prompt.md`. For orchestration, see `agents/vscode-chat-participants/feedback.md`.

---

---

## Section Constraints

| Section | Constraint |
|---------|-----------|
| 1. Accuracy Metrics | Percentages = valid + FP + won't fix (should sum to ~100%) |
| 2. FP Analysis | Only show categories with ≥1 finding. Flag >30% FP rate. |
| 3. Per-Report | One row per ticket. Sort by date (newest first). |
| 4. Prevention | Only generate if FP rate >30% OR same FP reason across 2+ reports. |
| 5. Trends | Only calculate with 3+ reports. Group by month if multi-month data. |

---

## Output Location

`reports/feedback/accuracy-report.md`
